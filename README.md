# Sales-Analytics-Dashboard
An interactive Power BI dashboard for analyzing shipment, sales, product performance, geographic distribution, and year-over-year business trends using dynamic KPIs and visualizations.

## 🎯 Objectives

- Monitor key business performance indicators (KPIs)
- Analyze sales and profit trends
- Compare business performance across different dimensions
- Identify top-performing and low-performing segments
- Provide interactive filtering for detailed analysis

## 🚀 How to Use

1. Download or clone this repository.
2. Open `Project.pbix` in Microsoft Power BI Desktop.
3. Refresh the dataset if necessary.
4. Explore the dashboard using the available filters and slicers.
